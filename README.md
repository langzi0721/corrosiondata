# Data Driven Prediction of Crevice Corrosion Failure in Stainless Steel

This repository contains the experimental data and model code for the paper *Data Driven Prediction of Crevice Corrosion Failure in Stainless Steel*, which supports the reproduction of data analysis and prediction results for crevice corrosion failure in stainless steel.


## 1. Repository Structure
- `data1/`: Raw experimental data (collected from 400 relevant publications in the LLM)
- `data2/`: Preprocessed & cleaned core dataset (directly used for the analysis in this paper)
- `code/`: Code scripts corresponding to different chapters/sections of the paper
